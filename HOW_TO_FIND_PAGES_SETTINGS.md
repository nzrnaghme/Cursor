# How to Find GitHub Pages Settings

## Step-by-Step Instructions

1. **You're already in the right place!** You're in the repository Settings page.

2. **Look at the LEFT SIDEBAR** - you'll see a list of options under "Code and automation"

3. **Scroll down** in the left sidebar until you see:
   - Actions
   - Models (currently selected - highlighted in blue)
   - Webhooks
   - Copilot
   - Environments
   - Codespaces
   - **Pages** ← **CLICK THIS ONE!**

4. **Click on "Pages"** in the left sidebar

5. Once you click "Pages", the main content area will show:
   - **Source** section (where you select the branch)
   - **Custom domain** section (where you enter `melodynazar.com`)
   - **Enforce HTTPS** checkbox (this is what you need to check!)

## What You'll See After Clicking "Pages"

1. **Source** section:
   - Branch: `gh-pages` (or `main`)
   - Folder: `/ (root)`

2. **Custom domain** section:
   - Input field: `melodynazar.com`
   - Save button

3. **Enforce HTTPS** section:
   - Checkbox: ☐ Enforce HTTPS
   - **THIS IS THE ONE YOU NEED TO CHECK!**

## Quick Action

**Just click "Pages" in the left sidebar, then check the "Enforce HTTPS" checkbox!**
